=====================================
INSTALACIÓN DEL PROYECTO
=====================================

Requisitos

- Python 3.11 o superior.
- Servidor MySQL/MariaDB en ejecución (por ejemplo, XAMPP).
- Base de datos creada y scripts SQL ejecutados.

IMPORTANTE

Durante la instalación de Python marcar:

[X] Add Python to PATH

La preparación de la base de datos y la ejecución
de los scripts SQL se encuentran explicadas en
la Sección 6 - Manual de Usuario del PDF adjunto.

-------------------------------------

Primer uso

1) Verificar que el servidor MySQL/MariaDB esté iniciado.
2) Confirmar que la base de datos haya sido creada.
3) Ejecutar setup.bat.
4) Esperar a que finalice la instalación.
5) Ejecutar ejecutar.bat.

-------------------------------------

Dependencias

mysql-connector-python==9.7.0

(Se instalarán automáticamente).

-------------------------------------

Si aparece un error indicando que Python
no está instalado, descárguelo desde:
