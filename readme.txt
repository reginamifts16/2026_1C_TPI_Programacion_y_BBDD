=====================================
INSTALACION DEL PROYECTO
=====================================

Requisitos
- Python 3.11 o superior.

IMPORTANTE
Durante la instalacion de Python marcar:
[X] Add Python to PATH

-------------------------------------

Primer uso
1) Ejecutar setup.bat
2) Esperar a que finalice la instalacion.
3) Ejecutar ejecutar.bat

-------------------------------------

Dependencias
mysql-connector-python==9.7.0
(se instalaran automaticamente)

-------------------------------------

Si aparece un error indicando que Python
no esta instalado, descarguelo desde:

https://www.python.org/downloads/
